# BlogApp
This is a blog app where users can read, write and like blogs.<br>
Users can write a blog within 1500 characters.

# Languages / Tech used
- Javascript 
- NodeJS
- MongoDB
- React
- Bootstrap

# Features
- Authentication with Google
- Reading Blogs
- Writing Blogs
- Like option
- REST APIs
